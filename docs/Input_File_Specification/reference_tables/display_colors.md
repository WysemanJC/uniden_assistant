# Display Colors Reference (164-Color Palette)

Complete color palette for scanner display customization. Colors are referenced by name (case-insensitive) in **DispColors** records, and the device maps them to hex RGB values for rendering.

[← Back to Main Index](../README.md)

---

## Used By

This reference table is used by the following record types:
- [DispColors](../record_types/scanner_profile_records.md#dispcolors) — Color configuration for UI elements

Related: [Display Options Reference](display_options.md) — Display area sizing, layout modes, and color usage patterns

---

## Format Specifications

- **Storage Format**: Color names are stored as strings (case-insensitive)
- **Rendering Format**: Device converts names to hex RGB triplets (e.g., `ffffff`)
- **Usage**: Typically specified in TextColor/BackColor pairs for UI elements

## Commonly Used Colors

Standard colors frequently used across display modes:

| Color Name | Hex Code | Typical Usage |
|---|---|---|
| **Black** | `000000` | Default background |
| **White** | `ffffff` | Standard text, options, status |
| **Orangered** | `ff4600` | System-level highlights |
| **Darkorange** | `ff8800` | Department-level highlights |
| **Gold** | `ffd600` | Channel-level highlights |
| **Orange** | `ffa100` | General accent |
| **Darksalmon** | `e79473` | Extended options |

---

## Usage Notes

- **Case-Insensitive**: Color names can be referenced in any case (e.g., "White", "white", "WHITE")
- **Color Pairs**: UI elements typically specify both TextColor and BackColor
- **Custom Colors**: All 164 colors are available for custom display configurations
- **Text Reversal**: Some UI elements (F button, Soft Keys) reverse text/background for emphasis
